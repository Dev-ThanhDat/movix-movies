# Movix Movies website

> Đây là dự án thiết kế website phim và chương trình truyền hình ở phía Frontend.

### Công nghệ sử dụng:

- ReactJS, SASS, Tailwind CSS, React Icons, Axios, Redux Toolkit, React Router Dom, swiper.

- **API**: Tích hợp api The Movie Database (TMDB) để truy xuất dữ liệu về phim và chương trình truyền hình.

### Chức năng:

- Ở trang chủ, bạn có thể chọn phim theo ngày hoặc chọn phim, chương trình truyền hình.

- Trang chi tiết phim hoặc chương trình truyền hình.

- Trang khám phá phim hoặc chương trình truyền hình.

- Tìm kiếm phim hoặc chương trình truyền hình.

- Sử dụng Axios để gọi API.

- Quản lý dữ liệu được API trả về khi sử dụng Axios bằng Redux Toolkit.
